# Heart-Disease-Prediction

### ❤️ Heart Disease Prediction

## 🎯 Task Objective
The objective of this project is to build a machine learning model that predicts whether a person is at risk of heart disease based on their health-related data. The model helps in early detection using classification techniques.

---

## 📊 Dataset Used
- Dataset: Heart Disease UCI Dataset  
- Source: Kaggle  
- Type: Structured medical dataset  

The dataset includes features such as age, blood pressure, cholesterol, chest pain type, and other clinical parameters.

---

## 🤖 Models Applied
The following machine learning models were used for classification:

- Logistic Regression  
- Decision Tree Classifier  

These models were trained to classify whether a patient has heart disease (1) or not (0).

---

## 📈 Key Results and Findings
- Logistic Regression provided stable and interpretable results  
- Decision Tree helped in understanding feature-based decision rules  
- Model performance was evaluated using:
  - Accuracy Score  
  - Confusion Matrix  
  - ROC Curve and ROC-AUC Score  

### Key Insights:
- Certain features like chest pain type, cholesterol, and maximum heart rate significantly influence prediction  
- The model successfully distinguishes between high-risk and low-risk patients  
- ROC curve shows good classification performance
